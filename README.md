# Bank Management System
A Python-based Bank Management System that allows users to
create accounts, deposit money, withdraw funds, and check balance.
## Features
     NEW ACCOUNT
     DEPOSIT AMOUNT
     WITHDRAW AMOUNT
     BALANCE ENQUIRY
     ALL ACCOUNT HOLDER LIST
     CLOSE AN ACCOUNT
     MODIFY AN ACCOUNT
     EXIT

## Technologies Used
- Python
- File Handling
- OOP Concepts

## How to Run
1. Clone the repository
2. Run `python main.py`
